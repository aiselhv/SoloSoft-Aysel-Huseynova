# CareerPilot

---

## Proje Hakkında

![Ürün Tanıtım Görseli](careerpilot.jpeg)

**Proje Tanımı:** 
CareerPilot, kullanıcıların kariyer gelişim süreçlerini planlayıp takip edebileceği kişiselleştirilmiş bir Kariyer Yönetim Platformudur. Platforma kayıt olan kullanıcılar kariyer hedeflerini ve mevcut yetkinliklerini sisteme tanımlayabilir, günlük görev listeleri oluşturabilir ve 3 ile 6 aylık kariyer yol haritalarını görüntüleyebilir. Skill Matrix sayesinde öğrenilen beceriler anlık olarak güncellenebilir; günlük ve haftalık ilerleme grafikleri ile hedefe ne kadar yaklaşıldığı yüzde olarak takip edilebilir. Görev tamamlama, yetenek seviyesi güncelleme ve Dark/Light Mode gibi özelliklerle CareerPilot, her kullanıcının kendi kariyer yolculuğunu yönetebileceği akıllı bir platform sunmaktadır.

**Proje Kategorisi:** 
Kariyer Yönetimi / Kişisel Gelişim

**Referans Uygulama:** 
[LinkedIn Learning](https://www.linkedin.com/learning)

## Projeyi Klonlama ve Düzenleme

### Repoyu Klonlama

```bash
git clone https://github.com/repo-kullanici-adi/careerpilot.git
```

### Projeyi Düzenleme

Kendi reponuza ekledikten sonra projeyi kendi bilgilerinizle güncellemeniz gerekmektedir:

1. **Proje Bilgilerini Güncelleme:**
   - `Readme.md` dosyasındaki proje adı, grup adı, ekip üyeleri
   - Proje tanımı ve kategorisi
   - Referans uygulama bilgisi

2. **Gereksinimleri Ekleme:**
   - `Gereksinim-Analizi.md` dosyasına kendi gereksinimlerinizi ekleyin
   - Her ekip üyesi için bir klasör oluşturun (örn: `Aysel-Huseynova/`)
   - Her ekip üyesinin klasörüne gereksinim dosyası ekleyin (örn: `Aysel-Huseynova/Aysel-Huseynova-Gereksinimler.md`)
   - Gereksinim sayılarına dikkat edin (Gereksinim-Analizi.md dosyasındaki kurallara göre)

3. **Dokümantasyonu Güncelleme:**
   - Tüm dokümantasyon dosyalarını kendi projenize göre düzenleyin
   - Her ekip üyesinin klasörüne görev dosyalarını ekleyin:
     - `Aysel-Huseynova/Aysel-Huseynova-Rest-API-Gorevleri.md`
     - `Aysel-Huseynova/Aysel-Huseynova-Web-Frontend-Gorevleri.md`
     - `Aysel-Huseynova/Aysel-Huseynova-Mobil-Frontend-Gorevleri.md`
     - `Aysel-Huseynova/Aysel-Huseynova-Mobil-Backend-Gorevleri.md`
   - Ekip üyelerinin görevlerini güncelleyin
   - API endpoint'lerini ve açıklamalarını kontrol edin

4. **Değişiklikleri Kaydetme:**
   ```bash
   git add .
   git commit -m "Proje bilgileri güncellendi"
   git push origin main
   ```
